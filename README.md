# Hospital Data Analysis Dashboard - Power BI

## Project Overview

This project is a Hospital Data Analysis Dashboard developed using Microsoft Power BI.
The dashboard helps visualize hospital data such as patient admissions, department performance, and overall hospital operations.

## Project Type

This was a **team project**, where I worked as the **team leader** and coordinated the development of the dashboard.

## Tools Used

* Microsoft Power BI
* Power Query
* Microsoft Excel

## Key Features

* Patient admission analysis
* Department-wise performance tracking
* Doctor and treatment statistics
* Interactive filters and visualizations

## Dashboard Insights

The dashboard helps users:

* Monitor hospital patient trends
* Analyze department performance
* Track hospital operational data
* Support data-driven healthcare decisions
